Editor Preferences

The following Counterclockwise preferences can be set on the Counterclockwise > Editor preference page and its sub-pages.

Option Description Default
Highlight matching brackets

If enabled, whenever the cursor is next to a parenthesis, bracket or curly braces, its opening or closing counter part is highlighted.

On

Displayed tab width

This option allows you to set the displayed tab width for text editors.

2

On file launch, switch the REPL to the file namespace (calling 'in-ns)

When a Clojure file is loaded into the REPL, this option will automatically call the in-ns function to allow direct calling of functions from the file just loaded, without the need to prefix the functions with the namespace prefix.

On

Start new editors in Strict Structural editing mode

When a Clojure file editor is started, should the Structural editing mode be set by default to "Default" (a mode which does not get in your way when you edit), or "Strict" (a mode which does its best to prevent you from breaking the structure of the code) ?

Off

Enable Clojure Support For a Project

For a New Project

  • Switch to the Java Perspective via menu Window > Open Perspective > Other > Java
  • Select menu File > New > New Clojure Project
  • Enter a project name
  • Click Finish

For an Existing Java Project

  • Display the context menu of the Java project node in the Project Explorer.
  • Select action Enable/Disable Clojure Language Support.
  • You should have a popup feedback on the correct enabling/disabling of Clojure support for this project.

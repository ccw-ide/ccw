Creating Clojure Files

Clojure files must be located in normal Java source directories. If you place a file with the .clj extension outside of a Java source directory, Eclipse will not see it on the classpath of the project, and it will not be possible to load it via the REPL or to select it as an automatically loaded file in the launcher customization wizard.

  • You can use File > New > Clojure File to create a Clojure file
  • Inside the Java source directory, follow the classical Clojure conventions to place your file, according to the namespace it belongs to. (e.g. you want to create namespace 'tests.clojure.first, then create Java package "tests.clojure", and create Clojure file first.clj in it).
